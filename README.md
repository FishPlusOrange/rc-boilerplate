A React components boilerplate.

## Start

```bash
# get boilerplate
git clone https://github.com/FishPlusOrange/rc-boilerplate.git && cd rc-boilerplate

# install
make install

# use storybook for document and development
make doc

# use with npm link for development
make dev

# test
make test

# build
make build

# release
make release
```
